Electronic Arts® Fsh format plugin for Adobe® Photoshop®


Installation

32-bit Plugin, Photoshop 6 or later

To install place FshFormat.8bi into Photoshop's Plug-In directory, typically "C\Program Files\Adobe\Photoshop <version>\Plug-Ins\File Formats" (or "C\Program Files (x86)\Adobe\Photoshop <version>\Plug-Ins\File Formats" if on a 64-bit OS).

The Microsoft Visual C++ 2010 x86 SP1 Redistributable is required to run the 32-bit plugin.

64-bit Plugin, Photoshop CS4 or later

To install place FshFormat64.8bi into Photoshop's Plug-In directory, typically "C\Program Files\Adobe\Photoshop <version>\Plug-Ins\File Formats"

The Microsoft Visual C++ 2010 x64 SP1 Redistributable is required to run the 64-bit plugin.
